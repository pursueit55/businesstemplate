

# Business Website Base Template

A clean, professional, fully responsive business website template built with pure HTML, CSS, and JavaScript. Designed as a reusable foundation for small business clients — swap in colors, content, and photos for any industry.

## Live Preview
> Open `business-template.html` in any browser — no installation required.

## Features
- Fixed navbar with scroll effect and mobile hamburger menu
- Hero section with stats bar
- About section with photo placeholder
- Services grid (6 cards) with hover effects
- Photo gallery with overlay
- Testimonials section
- Contact form with validation feedback
- Full footer with navigation columns
- Scroll reveal animations throughout
- Fully mobile responsive

## Built With
- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (no frameworks or dependencies)

## How to Customize
1. Open the file in Notepad or any code editor
2. Change the CSS variables at the top of the `<style>` block to update colors
3. Replace all placeholder text with client content
4. Swap photo placeholder divs with real `<img>` tags
5. Upload to any web host

## Color Customization
All colors are controlled by CSS variables at the top of the stylesheet:
```css
--navy:  #0A1628;   /* Primary dark color */
--blue:  #2D7DD2;   /* Accent color */
```
Change these two values and the entire site rebrands instantly.
